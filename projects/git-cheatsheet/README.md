# Interactive Git Cheatsheet

A modern, interactive guide for Git commands and concepts. Built with semantic HTML, BEM-compliant CSS, and vanilla JavaScript.

## Features
- Interactive accordion interface for commands.
- Dark/Light theme design with high contrast.
- BEM-compliant class structure.
- Responsive layout for all devices.
